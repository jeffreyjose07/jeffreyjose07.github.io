üìä What Data We Collect

We collect minimal, anonymized data to understand how the website is used:

  • Page views - Which pages are visited
  • Referrers - Where visitors came from (search engines, social media)
  • Country - Approximate location (country level only)
  • Browser type - General browser information
  • Screen size - To improve mobile experience

We do NOT collect:

  • IP addresses (automatically discarded)
  • Personal information or user accounts
  • Precise location data
  • Search queries or form data
  • Cross-site tracking data
  • Behavioral profiles

GoatCounter is an open-source, privacy-friendly analytics platform that doesn't use cookies or track users across sites.

üéØ How We Use This Data

The analytics data helps us:

  • Understand popular content - Which blog posts are most helpful
  • Improve performance - Optimize for common screen sizes and browsers
  • Track growth - See if the site is reaching more people

Data is never sold, shared with third parties, or used for advertising.

üìã Technical Implementation

For developers and security researchers, here's how analytics are implemented:

// Analytics script (only on page load): <script data-goatcounter="https://jeffreyjose.goatcounter.com/count" async src="//gc.zgo.at/count.js"></script> // View count API (public, no auth): fetch('https://jeffreyjose.goatcounter.com/counter/[path].json')

The implementation:

  • Runs only on page load (no continuous tracking)
  • Uses public API endpoints for view counts
  • Respects DNT headers and reduced motion preferences
  • Falls back gracefully if blocked

üåü Why This Matters

Privacy-first analytics demonstrates that you can understand your audience without compromising their privacy. This approach:

  • Builds trust through transparency
  • Reduces legal complexity (no GDPR consent needed)
  • Improves performance (no heavy tracking scripts)
  • Respects users as humans, not data points

Every website should consider privacy-first analytics. Analytics can be useful without being invasive.
